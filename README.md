# HackerRank C Solutions 🧩

> Solutions to HackerRank problems solved in C — organized by topic and difficulty for easy reference and learning.


---

## 🚀 Getting Started

### Prerequisites
- GCC compiler installed
- Any Linux/macOS/Windows terminal (or use an online compiler)

### Running a Solution

```bash
gcc solution.c -o solution
./solution
```

---

## 🧠 About

This repository contains my personal solutions to the [HackerRank C Path](https://www.hackerrank.com/domains/c). Each solution is written in standard C and aims to be clean, readable, and well-commented.

---

## 📌 Notes

- Solutions are written for **learning purposes**
- Each file is named after the corresponding HackerRank problem
- Feel free to open an issue or PR if you find a better approach!
